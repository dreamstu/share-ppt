# share-ppt

### based on reveal.js

一些ppt分享，ppt基于reveal.js生成


### how

#### 完整开发环境

1. `git clone https://github.com/dreamstu/share-ppt.git`
2. `cd share-ppt`
3. `npm install`
4. `grunt serve`


#### 仅查看

1. `git clone https://github.com/dreamstu/share-ppt.git`
2. `cd share-ppt`
4. open the index.html in broswer.
